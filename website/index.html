🦍 Chrome Extension

AI活用度を
見える化する

Googleカレンダーから日報を自動生成し、
各予定のAI活用度を記録。チームの業務効率化を加速します。

GitHub で見る 機能を見る →

主な機能

AI活用度の測定から分析レポートまで、業務効率化に必要なすべてを提供

📅

Googleカレンダー連携

当日の予定を自動取得。複数カレンダーに対応し、チェックボックスで対象予定を絞り込めます。

🤖

AI活用度記録

各予定に対してAI活用有無と活用率を記録。自動推定機能でキーワードベースの初期判定も行います。

📊

ダッシュボード

日別・週別・月別のサマリーを表示。AI活用時間の推移グラフとカレンダー別ランキングで傾向を把握。

✨

Gemini AIレポート

Gemini APIを活用し、週次・月次のAI活用レポートを自動生成。具体的な改善提案を受けられます。

📝

スプレッドシート保存

日報データをGoogleスプレッドシートへワンクリック保存。DailySummary・WeeklyReport・MonthlyReportを自動管理。

🔄

定期レポート自動生成

chrome.alarmsにより、毎週月曜・毎月1日にレポートを自動生成。通知でお知らせします。

使い方

3ステップでAI活用度の記録を開始

1

インストール

GitHubからソースを取得し、Chrome拡張として読み込み。Google Cloud設定で認証を有効化。

2

予定を確認

サイドパネルを開くと当日の予定が自動表示。各予定のAI活用度をスライダーで記録。

3

日報を送信

ワンクリックでスプレッドシートに保存。ダッシュボードで推移を確認し、AIレポートで改善。

AI活用度の可視化を始めよう
