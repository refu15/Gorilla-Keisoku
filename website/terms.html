利用規約

最終更新日: 2026年2月18日

本利用規約（以下「本規約」）は、Gorilla Keisoku（以下「本拡張機能」）の利用条件を定めるものです。本拡張機能をインストールまたは使用することにより、本規約に同意したものとみなします。

第1条（定義）

  • 「本拡張機能」とは、「AI活用度測定 - 日報作成アシスタント」Chrome拡張機能を指します
  • 「ユーザー」とは、本拡張機能をインストールし使用する個人または法人を指します
  • 「本サービス」とは、本拡張機能が提供するすべての機能を指します

第2条（サービスの概要）

本拡張機能は以下のサービスを提供します：

  • Googleカレンダーの予定データの取得と表示
  • 各予定に対するAI活用度の記録機能
  • 日報データのGoogleスプレッドシートへの保存
  • AI活用度のダッシュボード表示と分析
  • Gemini APIを利用したAIレポート生成
  • 定期的なレポート自動生成（アラーム機能）

第3条（利用条件）

本拡張機能を利用するには以下の条件を満たす必要があります：

  • Google Chrome ブラウザを使用していること
  • 有効なGoogleアカウントを保有していること
  • Google Calendar API および Google Sheets API へのアクセスを許可すること
  • Gemini APIの利用にはGemini APIキーが必要です（任意）

第4条（ユーザーの責任）

ユーザーは以下の事項について責任を負います：

  • Googleアカウントの認証情報の適切な管理
  • APIキーおよびスプレッドシートIDの適切な管理
  • スプレッドシートに保存されるデータの管理
  • 本拡張機能の適法な利用
  • 組織内利用の場合、所属組織のポリシーに準拠した使用

第5条（禁止事項）

ユーザーは以下の行為を行ってはなりません：

  • 本拡張機能のソースコードの不正な改変や再配布
  • 本拡張機能を利用した不正アクセスやスパム行為
  • 他のユーザーのデータへの不正アクセス
  • Google APIの利用規約に違反する行為
  • 法令に違反する目的での使用

第6条（知的財産権）

本拡張機能に関する知的財産権は、開発者に帰属します。本拡張機能のソースコードはGitHubで公開されていますが、利用にあたっては本規約およびリポジトリのライセンスに従う必要があります。

第7条（免責事項）

本拡張機能は「現状有姿」で提供されます。

  • 本拡張機能の利用により生じた損害について、開発者は一切の責任を負いません
  • データの正確性、完全性、信頼性について保証しません
  • Gemini APIによる分析結果の正確性について保証しません
  • Google APIやChrome APIの仕様変更による影響について責任を負いません
  • サービスの中断、停止、終了による損害について責任を負いません
  • ユーザーのデータ損失について責任を負いません

第8条（サービスの変更・停止）

開発者は、事前の通知なく、本拡張機能の機能を変更、追加、または削除する権利を有します。また、やむを得ない事情により、サービスの全部または一部を停止する場合があります。

第9条（個人情報の取り扱い）

ユーザーの個人情報の取り扱いについては、プライバシーポリシーに定めるとおりとします。

第10条（規約の変更）

開発者は、必要に応じて本規約を変更することができます。変更後の利用規約は、本ページに掲載した時点から効力を生じるものとします。変更後に本拡張機能を継続して使用した場合、変更後の規約に同意したものとみなします。

第11条（準拠法・管轄）

本規約は日本法に準拠して解釈されるものとします。本拡張機能に関連する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

第12条（お問い合わせ）

本規約に関するご質問がある場合は、GitHub Issuesよりお問い合わせください。
